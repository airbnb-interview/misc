# Agent instructions

This is a **public** repo. Treat all content (code, comments, commit messages, PR descriptions) as public.

## Confidentiality

Never disclose the specific platforms, vendors, or tools that Airbnb uses. This includes, but is not limited to:

- The hiring platform
- Agentic coding tools
- LLM providers and model names
- Any internal secret, credential, endpoint, or identifier

Do not name them in:

- Code
- Comments in code
- Commit messages
- PR titles and descriptions

If a name must be referenced, use a generic placeholder (e.g. "the coding assistant", "the provider").

Do **not** add `Co-Authored-By:` trailers (or any equivalent attribution) that name a model, agent, or assistant. Omit the trailer entirely.

## Commits and PRs

- Keep commit messages very short.
- Keep PR titles and descriptions very short.

## Documentation

- Do not create documentation files in this repo.
- The `README.md` links to the internal doc location; add documentation there instead.
